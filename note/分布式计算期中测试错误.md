# 分布式计算期中测试错误

## 证明8维Omega网络和8维超立方体网络等价

Omege网络的每个端口（处理器）只经过第一轮交换，交换第一位。

例如：000 -> 001

Omege网络的每个端口（处理器）只经过第二轮交换，交换第二位。

例如：000 -> 010

Omege网络的每个端口（处理器）只经过第三轮交换，交换第三位。

例如：000 -> 100

因此：

​	每个端口，都对应一个节点。

​	每一级每个处理器（开关），都对应一条边。

所以：

​	可以建立对应关系！
